# dotfiles

## 目的
PCお引越ししなくてはならないときに負荷無く同じ環境を作れるようにする
## 仕様(理想)
cloneしてスクリプト叩いたら引越し先で同じ環境が出来上がる

# 構成について
* git/
    * git関連
* zsh/
    * zsh関連
* manual/
    * スクリプトで解決できないような手続きをここに入れておきます
